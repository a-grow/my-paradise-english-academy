# 🌴 My Paradise English Academy

Website for **My Paradise English Academy** — an online English school for Taiwanese students, run by Aaron & Shirley.

## About
We offer group and one-on-one online English classes using the Oxford Discover curriculum. Our site features a class schedule, blog, and parent/student portal.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- SheetDB (Google Sheets as backend for blog & schedule)

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment
Deployed via GitHub Actions to GitHub Pages automatically on push to `main`.

## Structure
```
src/
  components/     # Reusable UI components
  pages/          # Page components (Home, Portal, Kids World)
  hooks/          # Custom React hooks
  lib/            # Utilities and helpers
```
